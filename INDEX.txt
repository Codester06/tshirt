QUICK START (3 STEPS)
================================================================================

Step 1: Setup (5 minutes)
  bash quick-start.sh

Step 2: Customize (10 minutes)
  • Export workflow.json from ComfyUI
  • Edit prompts.txt with your designs
  • (Optional) Adjust config.json

Step 3: Generate (varies)
  python run.py                    # Generate
  python batch_process.py --zip    # Archive

================================================================================
COMMON COMMANDS
================================================================================

# Preview without generating
python run.py --dry-run

# Generate all designs (ComfyUI must be running)
python run.py

# Use custom server
python run.py --server http://192.168.1.100:8188

# Post-process and create archive
python batch_process.py --zip

# Push to GitHub
git add .
git commit -m "Add designs"
git push
